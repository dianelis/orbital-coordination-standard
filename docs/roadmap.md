# Roadmap

- Define core message types.
- Publish JSON Schemas.
- Provide valid examples.
- Add a validator CLI.
- Add a toy two-operator coordination simulation.

- Add invalid examples and conformance tests.

- Add neighborhood-level multi-agent scenarios.
- Add routing and service continuity metrics.
- Add audit report generation.
- Add visualizations for paper figures.

## v1.0 Candidate

- Freeze required fields.
- Publish conformance language.
- Add implementation guide for civil coordination hubs.
- Submit for external review by researchers, operators, and policy experts.
